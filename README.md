# Wyrmspan Utilities

Some of my Wyrmspan utilities. Includes:

- **Score sheet (Automa):** Digital score sheet for games versus the Automa. Scores can be submitted to the Google Sheets document.
- **Score sheet:** Digital score sheet for games of 2 to 5 players. Scores can be submitted to the Google Sheets document.
